# Projekat1 – Web aplikacija

## Opis projekta
Web aplikacija za online kupovinu karata za gradski prevoz.

Aplikacija omogućava:
- Registraciju korisnika
- Prijavu i odjavu
- Izmenu korisničkog profila
- Rad sa MySQL bazom
- AJAX komunikaciju sa serverom

## Tehnologije
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- AJAX

## Pokretanje projekta
1. Instalirati XAMPP
2. Kopirati projekat u htdocs folder
3. Pokrenuti Apache i MySQL
4. Importovati bazu u phpMyAdmin
5. Pokrenuti aplikaciju na:
   http://localhost/Projekat
